# Operator

Minecraft Operator plugin based on Maven.

Features:
- `/operator` opens a GUI for operators.
- Teleport to online players through a menu.
- List loaded plugins in a GUI.
- Restart a selected plugin through the GUI.
- Enabled plugins are shown in green, disabled plugins in red.
- `/operator tp <player>` teleports directly by command.
- `/operator tphere <player>` brings a player to you.
- `/operator plugin` opens the plugin list directly.
- `/operator plugin reload <name>` restarts a specific plugin directly.
- `/operator plugin enabled <name>` enables a plugin.
- `/operator plugin disabled <name>` disables a plugin.
